# 可能出现的问题
## 1.响应时间  
也就是代码中的 sleep(10) 这里指等待10毫秒 即1秒100次 如果网速不行或者手机不行 可以适当调整到100/200/300/400/500  
## 2.App版本/各地区活动问题  
可能会导致部分 depth(X) 不太正常 需要针对app当前情况进行调整
## 3.AutoJs版本问题
上面网址所提到的官网版本在小米/华为/腾讯应用商店均可直接下载 安全无毒 但是有收费功能  
网上有4.11版本的免费版本 但是可能存在一定风险 请大家自行甄别 本人就不在此贴网址了
